Παραδείγματα

Όταν εκτελέσετε αυτόν τον κώδικα, θα σας ζητηθεί να εισάγετε έναν ακέραιο αριθμό. Μετά την εισαγωγή σας, το πρόγραμμα θα τυπώσει τον αριθμό που εισαγάγατε.

c

#include  <stdio.h>
int main() {
    int number;
    // Ζητάμε τον χρήστη να εισάγει έναν ακέραιο αριθμό
    printf("Παρακαλώ εισάγετε έναν ακέραιο αριθμό: ");
    scanf("%d", &number);
    // Εκτυπώνουμε τον αριθμό που εισήγαγε ο χρήστης
    printf("Εισαγωγή: %d\n", number);
    return 0;
}

Παράδειγμα που εκτυπώνει όλους τους τύπους δεδομένων.

c

#include  <stdio.h>
int main() {
    int integerNumber = 42;
    float floatNumber = 3.14159;
    double doubleNumber = 2.71828;
    char character = 'A';
    char string[] = "Hello, World!";          
    printf("Ακέραιος: %d\n", integerNumber);
    printf("Δεκαδικός αριθμός (float): %f\n", floatNumber);
    printf("Δεκαδικός αριθμός (double): %lf\n", doubleNumber);
    printf("Χαρακτήρας: %c\n", character);
    printf("Συμβολοσειρά: %s\n", string);          
    return 0;
}

Παράδειγμα με επανάληψης for που χρησιμοποιείται για να εκτελέσει τον κώδικα εντός του βρόχου για τις τιμές του i από 1 έως 5. Κάθε φορά που η επανάληψη εκτελείται, ο αριθμός i αυξάνεται κατά 1 και τυπώνεται.

c

#include  <stdio.h>
int main() {
    // Χρησιμοποιούμε τη δομή επανάληψης for για να εκτυπώσουμε τους αριθμούς 1 έως 5
    for (int i = 1; i <= 5; i++) {
        printf("Αριθμός: %d\n", i);
    }
    return 0;
}
