Πίνακες

Πίνακας δεδομένων

data = [2, 4, 6, 8, 10];

Πρόσθεση Στοιχείων Πίνακα

sumValue = sum(data); % sumValue = 30

Υπολογισμός Max και Min

maxValue = max(data); % Ανώτατο όριο, maxValue = 10
minValue = min(data); % Κατώτατο όριο, minValue = 2

Δημιουργία γραμμικού πίνακα

vec = 1:5; % vec = [1, 2, 3, 4, 5]

Δημιουργία γραμμικού πίνακα με βήμα 2

vec = 1:2:9; % vec = [1, 3, 5, 7, 9]

Πίνακας με Τυχαίους Αριθμούς

randMat = rand(3, 4); % Δημιουργία πίνακα 3x4 με τυχαίους αριθμούς από [0, 1)

Γραφική Αναπαράσταση (Plotting)

```matlab
x = 0:0.1:10; % Δημιουργία πίνακα τιμών x
y = sin(x); % Υπολογισμός τιμών y
plot(x, y); % Δημιουργία γραφήματος συνάρτησης sin(x)
xlabel('x'); % Ετικέτα άξονα x
ylabel('sin(x)'); % Ετικέτα άξονα y
title('Γράφημα της συνάρτησης sin(x)'); % Τίτλος γραφήματος
```
