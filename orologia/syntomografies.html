Ακρωνύμιο Πλήρες ΌνομαΠεριγραφή
APIApplication Programming InterfaceΜια συλλογή εργαλείων και πρωτοκόλλων που επιτρέπει την αλληλεπίδραση μεταξύ διαφορετικών λογισμικών.
SDKSoftware Development KitΣύνολο εργαλείων και βιβλιοθηκών που διευκολύνουν την ανάπτυξη λογισμικού για μια συγκεκριμένη πλατφόρμα.
IDEIntegrated Development EnvironmentΕφαρμογή που παρέχει εργαλεία για τη συγγραφή, αποσφαλμάτωση και ανάπτυξη λογισμικού.
GUIGraphical User InterfaceΔιεπαφή χρήστη που επιτρέπει την αλληλεπίδραση με το λογισμικό μέσω γραφικών στοιχείων.
OOPObject-Oriented ProgrammingΣτυλ προγραμματισμού που βασίζεται στη χρήση αντικειμένων και κλάσεων.
SQLStructured Query LanguageΓλώσσα για τη διαχείριση και ανάκτηση δεδομένων από σχεσιακές βάσεις δεδομένων.
JSONJavaScript Object NotationΜορφή ανταλλαγής δεδομένων που χρησιμοποιείται κυρίως για την αποθήκευση και μετάδοση δεδομένων.
XMLeXtensible Markup LanguageΓλώσσα σήμανσης που χρησιμοποιείται για την αποθήκευση και μεταφορά δεδομένων.
HTMLHyperText Markup LanguageΓλώσσα σήμανσης για τη δημιουργία και δομή ιστοσελίδων.
CSSCascading Style SheetsΓλώσσα που χρησιμοποιείται για τον καθορισμό της εμφάνισης και του στυλ των ιστοσελίδων.
HTTPHyperText Transfer ProtocolΠρωτόκολλο για την μεταφορά δεδομένων στο διαδίκτυο.
HTTPSHyperText Transfer Protocol SecureΑσφαλής έκδοση του HTTP που χρησιμοποιεί κρυπτογράφηση για την προστασία των δεδομένων.
HTTP/2HyperText Transfer Protocol version 2Εκδοχή του HTTP που βελτιώνει την απόδοση με την υποστήριξη πολλαπλών ροών δεδομένων.
HTML5HyperText Markup Language 5Η τελευταία έκδοση της γλώσσας HTML με νέες δυνατότητες για πολυμέσα και εφαρμογές.
RESTRepresentational State TransferΑρχιτεκτονικό στυλ για την ανάπτυξη web υπηρεσιών βασισμένο σε HTTP.
DevOpsDevelopment and OperationsΠρακτική που συνδυάζει την ανάπτυξη λογισμικού και τις λειτουργίες για την επιτάχυνση της ανάπτυξης και της διαχείρισης.
JSON-LDJSON for Linking DataΜορφή JSON που χρησιμοποιείται για την ενσωμάτωση συνδέσεων δεδομένων στον ιστό.
XML-RPCXML Remote Procedure CallΠρωτόκολλο που χρησιμοποιεί XML για την αποστολή αιτημάτων και λήψη απαντήσεων σε απομακρυσμένες διαδικασίες.
XSLTExtensible Stylesheet Language TransformationsΓλώσσα για τη μετατροπή XML σε άλλες μορφές, όπως HTML.
URLUniform Resource LocatorΔιεύθυνση που υποδεικνύει την τοποθεσία ενός πόρου στο διαδίκτυο
SQLStructured Query LanguageΓλώσσα προγραμματισμού για τη διαχείριση και χειρισμό δεδομένων σε βάσεις δεδομένων
DNSDomain Name SystemΣύστημα που μεταφράζει τα ονόματα τομέα σε διευθύνσεις IP
IPInternet ProtocolΠρωτόκολλο που χρησιμοποιείται για τη δρομολόγηση των πακέτων δεδομένων στο διαδίκτυο
CSVComma-Separated ValuesΑπλή μορφή αρχείου κειμένου που χρησιμοποιεί κόμματα για τη διαχωρισμένη αποθήκευση δεδομένων πίνακα.
