Oρολογία


Όρος Αγγλικά Περιγραφή
Μεταβλητή Variable Μεταβλητή είναι μια θέση μνήμης στον υπολογιστή την οποία έχουμε δεσμεύσει και της έχουμε ορίσει ένα όνομα. Το περιεχόμενο της συγκεκριμένης θέσης μνήμης είναι η τιμή της συγκεκριμένης μεταβλητής.
Συντακτικά λάθη Ονομάζουμε τα σφάλματα που σχετίζονται με τη σύνταξη μιας γλώσσας προγραμματισμού.
Αλγόριθμος Algorithm Αλγόριθμο ονομάζουμε ένα σύνολο εντολών (οδηγιών) που, αν εκτελεστούν με ακρίβεια, οδηγούν στην πραγματοποίηση μιας εργασίας ή την επίλυση ενός προβλήματος.
Γλώσσα υψηλού επιπέδο Η γλώσσα υψηλού επιπέδου θεωρείται η γλώσσα εκείνη που είναι αρκετά περιγραφική και είναι πιο κοντά στην ανθρώπινη γραπτή γλώσσα.
Γλώσσα χαμηλού επιπέδου Η γλώσσα χαμηλού επιπέδου θεωρείται εκείνη που είναι πιο κοντά στη μηχανή.
Μεταγλωττιστής Compiler Ο μεταγλωττιστής μετατρέπει τον κώδικα από μια γλώσσα σε μια άλλη (συνήθως σε Γλώσσα μηχανής).
Διερμηνέας Interpreter Ο διερμηνέας μεταφράζει τον κώδικα κατά τη διάρκεια της εκτέλεσης του προγράμματος.
Σύνταξη Syntax Είναι οι κανόνες που πρέπει να ακολουθεί ο κώδικας σε μια συγκεκριμένη γλώσσα προγραμματισμού ώστε να είναι σωστά γραμμένος και να μπορεί να εκτελεστεί.
Συνάρτηση Function Είναι ένα τμήμα κώδικα που εκτελεί μια συγκεκριμένη εργασία όταν καλείται από ένα άλλο σημείο του προγράμματος. Συνήθως είναι ένα κομμάτι κώδικα που το χρησιμοποιούμε πολλές φορές μέσα στο πρόγραμμα.
Συμβολοσειρά String Είναι μια σειρά από χαρακτήρες (γράμματα, αριθμοί, σύμβολα).
Πίνακας Array Είναι μια δομή δεδομένων που αποθηκεύει μια συλλογή από στοιχεία του ίδιου τύπου, με κάθε στοιχείο να έχει ένα μοναδικό αριθμό ευρετηρίου (index).
Εξαίρεση Exception Είναι μια ανωμαλία που συμβαίνει κατά την εκτέλεση ενός προγράμματος και διακόπτει τη φυσιολογική ροή εκτέλεσης του προγράμματος.
Βρόχος Loop Είναι μια δομή ελέγχου ροής που επιτρέπει την επαναληπτική εκτέλεση ενός τμήματος κώδικα μέχρι να ικανοποιηθεί μια συγκεκριμένη συνθήκη.
Αντικείμενο Object Στον αντικειμενοστραφή προγραμματισμό, ένα αντικείμενο είναι μια συγκεκριμένη εμφάνιση μιας κλάσης που συνδυάζει δεδομένα (μεταβλητές) και λειτουργίες (μεθόδους).
Κλάση Class Είναι ένα πρότυπο από το οποίο δημιουργούνται αντικείμενα. Μια κλάση περιέχει μεταβλητές και μεθόδους που ορίζουν τις ιδιότητες και τη συμπεριφορά των αντικειμένων της.
Στοίβα Stack Η στοίβα λειτουργεί με τη μέθοδο Last In, First Out (LIFO)
Ουρά Queue Η ουρά λειτουργεί με τη μέθοδο First In, First Out (FIFO)
Κληρονομικότητα Inheritance Είναι η δυνατότητα μιας κλάσης να παίρνει χαρακτηριστικά (μεταβλητές και μεθόδους) μιας άλλης κλάσης και να προσθέτει τα δικά της.
Πολυμορφισμός Polymorphism Είναι η δυνατότητα των αντικειμένων να παρουσιάζουν διαφορετική συμπεριφορά μέσω της χρήσης των ίδιων ονομάτων μεθόδων.
