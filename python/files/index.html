Λίστες

Ανάγνωση (Read)

file = open("example.txt", "r")  # Άνοιγμα για ανάγνωση
content = file.read()
print(content)
file.close()

Ανάγνωση Γραμμών

readline(): Διαβάζει μία γραμμή τη φορά.
readlines(): Διαβάζει όλες τις γραμμές και τις επιστρέφει ως λίστα.

file = open("example.txt", "r")
line = file.readline()
lines = file.readlines()
file.close()

Εγγραφή σε Αρχείο (Write)

Ανοίγει το αρχείο για εγγραφή. Αν το αρχείο υπάρχει, διαγράφει το περιεχόμενο και ξεκινάει με νέο περιεχόμενο. Αν δεν υπάρχει, το δημιουργεί.

file = open("example.txt", "w")  # Άνοιγμα αρχείου για εγγραφή
file.write("Hello, World!")
file.close()

Κλείσιμο Αρχείου (Close)

Πρέπει πάντα να κλείνεις το αρχείο αφού τελειώσεις μαζί του, για να απελευθερωθούν οι πόροι.

file = open("example.txt", "r")
# Διαχείριση του αρχείου
file.close()

Χρήση του with για αυτόματο κλείσιμο

Η χρήση του with κλείνει αυτόματα το αρχείο όταν η εργασία ολοκληρωθεί.

with open("example.txt", "r") as file:
    content = file.read()
    print(content)
# Το αρχείο κλείνει αυτόματα εδώ

Προσθήκη σε Αρχείο (Append)

Η λειτουργία append χρησιμοποιείται για την προσθήκη δεδομένων στο τέλος ενός αρχείου.

file = open("example.txt", "a")  # Άνοιγμα αρχείου για προσθήκη
file.write("\nΠροσθήκη νέας γραμμής")
file.close()

Δημιουργία Νέου Αρχείου (Exclusive Creation)

Δημιουργεί το αρχείο και το ανοίγει για εγγραφή. Αν το αρχείο υπάρχει ήδη, θα προκληθεί σφάλμα.

file = open("newfile.txt", "x")  # Δημιουργία νέου αρχείου
file.write("This is a new file.")
file.close()

Ανάγνωση δυαδικού αρχείου

file = open("image.png", "rb")  # Δυαδική ανάγνωση
content = file.read()
file.close()

Εγγραφή σε δυαδικό αρχείο

file = open("newimage.png", "wb")  # Δυαδική εγγραφή
file.write(content)  # content μπορεί να είναι δεδομένα δυαδικής μορφής
file.close()

Text

Ανοίγει το αρχείο σε κειμενική μορφή (προεπιλογή). Συνήθως δεν χρειάζεται να προσθέσεις αυτό το mode, καθώς είναι η προεπιλεγμένη συμπεριφορά, αλλά μπορεί να χρησιμοποιηθεί συνδυαστικά με άλλες λειτουργίες όπως 'rt', 'wt', κ.λπ.

file = open("example.txt", "rt")  # Κειμενική ανάγνωση (ίδιο με το "r")
content = file.read()
file.close()

Ανάγνωση και Εγγραφή (Update)

file = open("example.txt", "r+")  # Ανάγνωση και εγγραφή
content = file.read()
file.write("\nAdding a new line after reading.")
file.close()

Δημιουργία και εγγραφή με ανάγνωση

file = open("example.txt", "w+")  # Δημιουργία/εγγραφή και ανάγνωση
file.write("Starting fresh with new content.")
file.seek(0)  # Μετακίνηση στην αρχή του αρχείου
print(file.read())  # Ανάγνωση του νέου περιεχομένου
file.close()

Προσθήκη και Ανάγνωση (Append and Read)

Ανοίγει το αρχείο για προσθήκη και ανάγνωση. Αν το αρχείο δεν υπάρχει, το δημιουργεί.

file = open("example.txt", "a+")  # Προσθήκη και ανάγνωση
file.write("\nAppending another line.")
file.seek(0)  # Μετακίνηση στην αρχή του αρχείου για ανάγνωση
print(file.read())
file.close()

Εγγραφή και Ανάγνωση (Write and Read)

Ανοίγει το αρχείο για εγγραφή και ανάγνωση. Διαγράφει το υπάρχον περιεχόμενο του αρχείου, αν υπάρχει.

file = open("example.txt", "w+")  # Εγγραφή και ανάγνωση
file.write("Overwriting content.")
file.seek(0)
print(file.read())
file.close()

Δημιουργία Νέου Αρχείου και Ανάγνωση (Exclusive Creation and Read)

Δημιουργεί το αρχείο για εγγραφή και ανάγνωση. Αν το αρχείο υπάρχει ήδη, προκαλεί σφάλμα.

file = open("newfile.txt", "x+")  # Δημιουργία νέου αρχείου για εγγραφή και ανάγνωση
file.write("This is a new file with read/write access.")
file.seek(0)
print(file.read())
file.close()
