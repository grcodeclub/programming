Python - Συμβολοσειρές

Η μέθοδος capitalize() κάνει το πρώτο γράμμα του string.

text = "grCode club"
print(text.capitalize())  # Εκτύπωση: GrCode club

Η μέθοδο title() κάνει capitalize κάθε πρώτο γράμμα κάθε λέξης στο string.

text = "grCode club"
print(text.title())  # Εκτύπωση: GrCode Club

Η μέθοδος upper() επιστρέφει τη συμβολοσειρά με κεφαλαία γράμματα.

text = "GrCode Club"
print(text.upper()) #Εκτύπωση 'GRCODE CLUB'

Η μέθοδος lower() επιστρέφει τη συμβολοσειρά με πεζά γράμματα.

text = "GrCode Club"
print(text.lower()) #Εκτύπωση 'grcode club'

Η μέθοδος strip() αφαιρεί οποιοδήποτε κενό διάστημα από την αρχή ή το τέλος.

text= "GrCode Club"
print(text.strip()) #Εκτύπωση 'GrCode Club'

Η μέθοδος swapcase() αλλάζει τα πεζά σε κεφαλαία και το αντίστροφο σε κάθε γράμμα του string.

text= "GrCode Club"
print(text.swapcase())  # Εκτύπωση: gRcODE cLUB

Multiline Strings

Μπορείτε να χρησιμοποιήσετε 3 διπλά εισαγωγικά ή 3 μεμονωμένα αποσπάσματα:

text = """Η Python είναι μια αντικειμενοστραφής γλώσσα προγραμματισμού (object-oriented)

προγραμματιστής ψάχνει για μια εύκολη στη χρήση γλώσσα"""
print(text)

Ελέγξτε μια συμβολοσειρά

Για να ελέγξουμε εάν μια συγκεκριμένη φράση ή χαρακτήρας υπάρχει σε μια συμβολοσειρά, μπορούμε να χρησιμοποιήσουμε τη λέξη-κλειδί in.

text = "GrCode Club"
print("Gr" in text) #Εκτύπωση True

Για να ελέγξουμε αν μια συγκεκριμένη φράση ή χαρακτήρας ΔΕΝ υπάρχει σε μια συμβολοσειρά, μπορούμε να χρησιμοποιήσουμε τη λέξη-κλειδί not in.

text = "GrCode Club"
print("Hello" not in text) #Εκτύπωση True
